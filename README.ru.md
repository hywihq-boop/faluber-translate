# 🌐 Faluber Translate — ИИ-переводчик веб-страниц

Расширение для Chrome, переводящее веб-страницы с помощью ИИ (OpenAI-совместимые API: DeepSeek, OpenAI, Qwen и др.). Основано на Manifest V3, выполняет перевод на уровне текстовых узлов DOM, поддерживая 50 целевых языков и 20 языков интерфейса.

## ✨ Возможности

- 🚀 **Перевод всей страницы в один клик** — нажмите иконку или `Alt+T`
- 🌍 **50 целевых языков** — ИИ-перевод для любых языковых пар
- ⚡ **Два режима** — Стандартный (сбалансированный) и Турбо (максимальная скорость)
- 🔑 **Поддержка нескольких API** — 10 встроенных провайдеров (DeepSeek, OpenAI, Groq, Qwen и др.)
- 💬 **Всплывающее объяснение по Ctrl** — наведите на слово + нажмите Ctrl для ИИ-объяснения
- 📋 **Панель перевода** — `Alt+Q` для панели ввода/вывода с двумя колонками
- 👆 **Просмотр оригинала при наведении** — наведите на переведённый текст
- ↩️ **Восстановление в один клик** — мгновенный возврат к исходной странице
- 📊 **Прогресс в реальном времени** — визуальная полоса прогресса
- 💾 **Умное кэширование** — кэш в памяти + постоянный кэш, TTL 1 час
- 🎨 **Тёмный интерфейс** — сворачиваемый плавающий виджет

## 📦 Установка

### 1. Загрузка расширения

1. Откройте Chrome/Edge и перейдите в `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель справа вверху)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `faluber translate`
5. Готово!

### 2. Настройка API

1. Нажмите иконку Faluber Translate на панели инструментов
2. Выберите провайдера из списка (DeepSeek, OpenAI и др.)
3. Введите ваш **API Key**
4. Нажмите **Проверить соединение** для проверки
5. Нажмите **Сохранить**

### 3. Начало перевода

- Откройте любую веб-страницу, нажмите иконку → **Перевести страницу**
- Или нажмите **`Alt+T`**

## 🛠️ Как это работает

### Процесс перевода

```
Пользователь запускает перевод
  → Content Script обходит DOM, собирает видимые текстовые узлы
  → Фильтры: проверка видимости, дедупликация CJK, мин. длина, дедупликация кэша
  → Сортировка по Y-координате, слияние соседних
  → Пакетная отправка в Service Worker (3–8 параллельных воркеров)
  → Service Worker вызывает AI API (OpenAI-совместимый формат)
  → Результаты возвращаются в Content Script для замены DOM
  → Полоса прогресса в реальном времени + уведомление о завершении
```

### Умная обработка текста

- Пропуск `<script>`, `<style>`, `<code>` и других неконтентных тегов
- Фильтрация чисел, URL, эмодзи и нетекстового содержимого
- Проверка `display:none`, `visibility:hidden` и других скрытых состояний
- Сохранение HTML-сущностей и специальных символов
- Соседние текстовые узлы объединяются перед отправкой

### Режимы перевода

| Параметр | Стандартный | Турбо |
|----------|------------|-------|
| Параллельность | 3 | 8 |
| Размер пакета | 400 симв. | 250 симв. |
| Вся страница | Нет (только viewport) | Да |
| Отслеживание скролла | Да | Нет |
| Отслеживание наведения | Да | Нет |
| MutationObserver | Да | Да |

### Система кэширования

- **В памяти**: `Map<оригинал, перевод>` для мгновенного поиска
- **Постоянный**: `chrome.storage.local`, до 2 000 записей, TTL 1 час
- **Автосброс**: каждые 30 секунд + при `beforeunload`
- **Автоочистка**: при смене целевого языка

### Ctrl+Объяснение (всплывающий пузырь)

- Наведите на слово + нажмите `Ctrl` (не удерживая)
- Или выделите текст + нажмите `Ctrl`
- Двухуровневая система: естественно-языковой промпт → анализ HTML
- Закрытие: ✕, `Esc` или клик в любом другом месте

## 🔒 Приватность

- Ваш API Key хранится локально в синхронизированном хранилище Chrome
- Все запросы на перевод идут напрямую из браузера к настроенному API-провайдеру
- Никаких сторонних серверов — ваши данные остаются между вами и API-провайдером
