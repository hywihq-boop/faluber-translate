核心功能

不只是翻译

文本节点级 DOM 替换 + AI 大模型驱动，翻译精准不破坏页面结构。

🌐

一键整页翻译

点击悬浮球或按 Alt+T，将整个网页翻译为你的目标语言。支持自动检测源语言。

🔍

Ctrl + 悬停解释

按住 Ctrl 将鼠标指向任意词汇，AI 即刻给出解释。品牌、术语、缩写都能识别。划词后按 Ctrl 解释整段文字。

⚡

双档翻译模式

标准模式平衡速度与消耗，极速模式 8 路并发 + 全页翻译，首批译文秒出。

🔑

多 API 管理

保存多个 API 配置，随时切换。支持 10 家主流大模型提供商，自动填入地址和拉取可用模型。

💬

20 种界面语言

插件界面支持 20 种语言，翻译目标语言覆盖 50 种。切换语言后缓存自动清除，适配多语用户。

📊

Token 用量统计

实时显示本页和全局 token 消耗，预估费用。支持缓存，重复内容不消耗 token。

翻译模式

两档可选，按需切换

在悬浮球详情面板中随时切换，设置持久化保存。

安装步骤

三步开始使用

1

下载插件

从 Chrome Web Store 安装，或下载 zip 包手动加载。

2

配置 API

点击插件图标 → 选择大模型提供商（DeepSeek/OpenAI/Groq 等 10 家）→ 填入 API Key → 测试连接。

3

开始使用

打开任意网页，点击右下角悬浮球翻译整页；或按住 Ctrl 悬停任意词汇即时解释。
