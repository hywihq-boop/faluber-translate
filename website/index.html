核心功能

不只是翻译

文本节点级 DOM 替换 + AI 大模型驱动，翻译精准不破坏页面结构。

🌐

一键整页翻译

点击悬浮球或按 Alt+T，将整个网页翻译为目标语言。文本节点级替换，不破坏页面结构。支持自动检测源语言。

🔍

鼠标指向 + Ctrl 解释

将鼠标指向任意词汇，单击 Ctrl 键，AI 即刻弹出气泡解释。划词后按 Ctrl 解释整段文字。不修改网页文本，零 DOM 污染。

⚡

双档翻译模式

标准模式（3 并发 + 视野优先）平衡速度与消耗。极速模式（8 并发 + 全页翻译）首批译文秒出。随时切换，持久化保存。

🔑

多 API 管理

保存多个 API 配置随时切换。内置 10 家大模型提供商预设地址，自动拉取可用模型。支持自定义接口。

💬

50 种翻译语言

提示词按目标语言本地化发送，零跨语言 token 浪费。界面支持 20 种语言，切换后缓存自动清除。

📊

Token 用量统计

实时显示本页和全局 token 消耗，预估费用。智能缓存，重复内容不消耗 token。

安装配置

三步开始使用

下载 → 配置 API → 开始翻译

1

下载并安装

下载 zip 解压，打开 chrome://extensions，开启开发者模式，加载已解压的扩展程序。

2

配置 API

点击插件图标，选择大模型提供商（推荐 DeepSeek），填入 API Key，测试连接，拉取可用模型。

3

开始翻译

打开任意网页，点击悬浮球翻译整页。鼠标指向词汇 + 单击 Ctrl 即刻弹出解释气泡。
