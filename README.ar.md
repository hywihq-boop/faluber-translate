# 🌐 Faluber Translate — ترجمة صفحات الويب بالذكاء الاصطناعي

إضافة لمتصفح Chrome تترجم صفحات الويب باستخدام الذكاء الاصطناعي (واجهات برمجة متوافقة مع OpenAI مثل DeepSeek وOpenAI وQwen وغيرها). مبنية على Manifest V3، تقوم بمعالجة النصوص على مستوى عقد DOM لترجمة سلسة عبر 50 لغة مستهدفة مع واجهة بـ 20 لغة.

## ✨ الميزات

- 🚀 **ترجمة الصفحة كاملة بنقرة واحدة** — انقر الأيقونة أو اضغط `Alt+T`
- 🌍 **50 لغة مستهدفة** — مدعومة بالذكاء الاصطناعي، تدعم أي زوج لغوي
- ⚡ **وضعان للترجمة** — قياسي (متوازن) وعالي (أقصى سرعة)
- 🔑 **دعم متعدد لواجهات API** — 10 إعدادات مسبقة لمزودين (DeepSeek وOpenAI وGroq وQwen وغيرها)
- 💬 **فقاعة Ctrl+شرح** — مرر فوق كلمة + اضغط Ctrl للحصول على شرح بالذكاء الاصطناعي
- 📋 **لوحة الترجمة** — اضغط `Alt+Q` للوحة ترجمة جانبية للإدخال/الإخراج
- 👆 **المرور لرؤية الأصل** — مرر فوق النص المترجم لرؤية المصدر
- ↩️ **استعادة بنقرة واحدة** — عد إلى الصفحة الأصلية في أي وقت
- 📊 **تقدم مباشر** — شريط تقدم مرئي أثناء الترجمة
- 💾 **تخزين مؤقت ذكي** — ذاكرة + تخزين دائم مع TTL لمدة ساعة
- 🎨 **واجهة داكنة نظيفة** — عنصر عائم قابل للطي

## 📦 التثبيت

### 1. تحميل الإضافة

1. افتح Chrome/Edge وانتقل إلى `chrome://extensions/`
2. فعّل **وضع المطور** (الزر في الأعلى يميناً)
3. انقر **تحميل إضافة غير مضغوطة**
4. اختر مجلد `faluber translate`
5. تم!

### 2. إعداد API

1. انقر أيقونة Faluber Translate في شريط الأدوات
2. اختر مزوداً من القائمة (DeepSeek، OpenAI، إلخ)
3. أدخل **مفتاح API** الخاص بك
4. انقر **اختبار الاتصال** للتحقق
5. انقر **حفظ**

### 3. ابدأ الترجمة

- افتح أي صفحة ويب، انقر الأيقونة ← **ترجمة الصفحة**
- أو اضغط **`Alt+T`**

## 🛠️ آلية العمل

### سير عملية الترجمة

```
يقوم المستخدم بتشغيل الترجمة
  → يتصفح Content Script شجرة DOM ويجمع عقد النصوص المرئية
  → المرشحات: التحقق من الظهور، إزالة تكرار CJK، الحد الأدنى للطول، إزالة تكرار التخزين المؤقت
  → الترتيب حسب الإحداثي Y، دمج المتجاورة
  → تجميع وإرسال إلى Service Worker (3–8 عمال متوازيين)
  → يستدعي Service Worker واجهة AI API (بتنسيق متوافق مع OpenAI)
  → تعاد النتائج إلى Content Script لاستبدال النص في DOM
  → شريط تقدم مباشر + إشعار بالإكمال
```

### معالجة ذكية للنصوص

- تخطي `<script>` و`<style>` و`<code>` والوسوم غير المحتواية الأخرى
- تصفية الأرقام البحتة والروابط والرموز التعبيرية والمحتوى غير القابل للترجمة
- التحقق من `display:none` و`visibility:hidden` وحالات الإخفاء الأخرى
- الحفاظ على كيانات HTML والرموز الخاصة
- دمج عقد النصوص المتجاورة (نفس العنصر الأب أو عناصر شقيقة) قبل الإرسال

### أوضاع الترجمة

| المعامل | قياسي | عالي (توربو) |
|---------|-------|-------------|
| التوازي | 3 | 8 |
| حجم الدفعة | 400 حرف | 250 حرف |
| الصفحة كاملة | لا (منطقة المشاهدة فقط) | نعم |
| كشف التمرير | نعم | لا |
| كشف المرور | نعم | لا |
| مراقب التغييرات | نعم | نعم |

### نظام التخزين المؤقت

- **في الذاكرة**: `Map<النص الأصلي، الترجمة>` للبحث الفوري
- **دائم**: `chrome.storage.local`، حتى 2,000 مدخلة، TTL ساعة واحدة
- **حفظ تلقائي**: كل 30 ثانية + عند `beforeunload`
- **مسح تلقائي**: عند تغيير اللغة المستهدفة

### Ctrl+شرح (الفقاعة العائمة)

- مرر فوق كلمة + اضغط `Ctrl` (نقرة وليس ضغطاً مستمراً)
- أو حدد نصاً + اضغط `Ctrl`
- نظام احتياطي من مستويين: موجه باللغة الطبيعية → فحص HTML
- أغلق بـ ✕ أو `Esc` أو النقر في أي مكان آخر

## 🔒 الخصوصية

- يُخزن مفتاح API محلياً في تخزين Chrome المتزامن
- جميع طلبات الترجمة تنتقل مباشرة من متصفحك إلى مزود API الذي قمت بإعداده
- لا توجد خوادم طرف ثالث — بياناتك تبقى بينك وبين مزود API الخاص بك
