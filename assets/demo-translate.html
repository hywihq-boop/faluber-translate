Faluber Translate

Faluber Translate is an open-source Chrome extension that uses AI to translate web pages. It replaces text nodes directly in the DOM without iframes or overlays, delivering a clean, native-feeling translation experience.

The extension supports 50 target languages and features 10 built-in API provider presets including DeepSeek, OpenAI, Groq, Qwen, and more. Its dual translation modes — Standard and Turbo — let you balance speed against API costs.

A unique Ctrl+Explain feature lets you hover over any word and tap Ctrl for an AI-powered explanation bubble. The floating translation panel (Alt+Q) provides side-by-side input and output for quick manual translations.

All processing happens client-side — your API key stays local, and translation requests go directly from your browser to your configured provider. No third-party servers involved.
