Faluber Translate

基于 AI 大模型的智能网页翻译插件。鼠标指向 + 单击 Ctrl 即时解释，Alt+T 一键翻译整页，支持 50 种语言，兼容 OpenAI 格式 API。

AI-powered web page translation extension. Hover + Ctrl for instant explanations, Alt+T for full-page translation. 50 languages, compatible with any OpenAI-format API.

核心功能
Core Features

不只是翻译

More Than Translation

文本节点级 DOM 替换 + AI 大模型驱动，翻译精准不破坏页面结构。支持 50 种目标语言，20 种 UI 语言。

Text-node-level DOM replacement + AI-driven translation. Accurate results without breaking page layout. 50 target languages, 20 UI languages.

🌐

一键整页翻译

One-Click Full-Page

点击悬浮球或按 Alt+T，将整个网页翻译为目标语言。文本节点级替换，不破坏页面结构。

Click the widget or press Alt+T to translate the entire page. Text-node-level replacement preserves page structure.

🔍

Ctrl + 智能解释

Ctrl + Explain

鼠标指向任意词汇 + 单击 Ctrl，AI 弹出气泡解释。划词后按 Ctrl 解释整段。不修改网页文本。

Hover over any word + tap Ctrl for an AI explanation bubble. Select text + Ctrl to explain a passage. Zero DOM pollution.

⚡

双档翻译模式

Dual Modes

标准模式（3 并发 + 视野优先）平衡速度与消耗。极速模式（8 并发 + 全页翻译）速度拉满。

Standard mode (3 concurrent, viewport-first) balances speed & cost. Turbo mode (8 concurrent, full-page) for maximum speed.

🔑

多 API 管理

Multi-API

内置 10 家提供商预设，保存多个 API 配置随时切换。自动拉取可用模型，支持自定义接口。

10 built-in provider presets. Save multiple API configs and switch anytime. Auto-fetch models. Custom endpoints supported.

📋

翻译面板 (Alt+Q)

Translation Panel

左右布局的浮动翻译面板，输入原文即时翻译。支持任意语言对，独立于页面翻译。

Side-by-side floating panel for instant input/output translation. Works independently from page translation.

💾

智能缓存

Smart Cache

内存 + 持久化双层缓存，1 小时 TTL。重复内容不消耗 token。语言切换自动清除缓存。

In-memory + persistent dual-layer cache with 1-hour TTL. No duplicate token spend. Auto-clears on language change.

安装配置
Setup

三步开始使用

Get Started in 3 Steps

下载 → 配置 API → 开始翻译

Download → Configure API → Translate

1

下载并安装

Download & Install

从 GitHub Releases 下载 zip，解压后打开 chrome://extensions，开启开发者模式，加载已解压的扩展程序。

Download the zip from GitHub Releases, unzip, open chrome://extensions, enable Developer mode, and load unpacked.

2

配置 API

Configure API

点击插件图标，选择提供商（支持 DeepSeek、OpenAI、Groq 等 10 家），填入 API Key，测试连接。

Click the extension icon, choose a provider (10 presets: DeepSeek, OpenAI, Groq, etc.), enter your API key, and test the connection.

3

开始翻译

Start Translating

打开任意网页，点击悬浮球翻译整页。鼠标指向词汇 + 单击 Ctrl 即刻弹出解释气泡。按 Alt+Q 打开翻译面板。

Open any webpage, click the widget to translate. Hover over a word + tap Ctrl for instant explanation. Press Alt+Q for the translation panel.
